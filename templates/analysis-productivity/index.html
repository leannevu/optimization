About This Project

Why I built this

As I transition from Computer Science to Data Science, I wanted to deepen my understanding of exploratory data analysis while building user-focused tools. I created this interactive EDA dashboard using Python, Flask, Pandas, and Matplotlib to present analysis in a way that feels more guided, visual, and accessible.

The goal was to bring together notebook-style reasoning and web interactivity — allowing users to reveal code, inspect results, and move through the analysis step by step instead of consuming the project as one long static report.
