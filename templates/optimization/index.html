About This Project

Why I built this

I built this project to strengthen my understanding of portfolio optimization while exploring how quantitative analysis can be turned into a more user-friendly interactive tool. Rather than keeping the work inside a notebook, I wanted to make the results easier to explore through a simple web interface.

The project combines financial modeling concepts with product thinking by translating analytical outputs into a clearer workflow: generate, interpret, and visualize an optimized portfolio in one place.
