About This Project

Why I built this

Inspired by my internship experience in agile teams, I built this SCRUM management tool to explore how a lightweight interface can improve sprint visibility, coordination, and backlog tracking.

I originally developed it using PHP and MySQL, then rebuilt it in Python with Flask after the site crashed. The rewrite gave me the opportunity to simplify the architecture and rethink the workflow from a product perspective.
